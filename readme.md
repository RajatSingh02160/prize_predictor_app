# Parking Price Predictor 

A Machine Learning web application that predicts parking prices dynamically based on parking conditions such as occupancy, queue length, and special day status.

This project was built using Flask, HTML/CSS, and Machine Learning to demonstrate an end-to-end ML deployment workflow.

---

## Features

- Dynamic parking price prediction
- Machine Learning model using Random Forest Regressor
- Feature selection for scalability
- Flask backend integration
- Responsive frontend using HTML & CSS
- User input validation
- Real-time prediction output

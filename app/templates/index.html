Dynamic Parking Price Predictor

This machine learning web application predicts parking prices dynamically based on occupancy, queue length, and special day conditions. The goal of this project is to demonstrate how AI and machine learning can help optimize real-time pricing systems and improve parking management efficiency.

This project was initially inspired by the Summer Analytics capstone project by IIT Guwahati and was later independently extended by Rajat into a complete end-to-end machine learning deployment application using Flask, HTML, CSS, and Random Forest Regression.
